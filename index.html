Qui suis-je

Selssabila Kiouaz is a cybersecurity engineer and full-stack developer from Algeria, currently pursuing an L3 in Mathematics-Computer Science at Université Savoie Mont Blanc (Chambéry, France), with a Master's in Cybersecurity Engineering from ESTIN Bejaia.

Selssabila Kiouaz est ingénieure en cybersécurité et développeuse full-stack algérienne, actuellement en L3 Mathématiques-Informatique à l'Université Savoie Mont Blanc (Chambéry), titulaire d'un Master 1 Cybersécurité de l'ESTIN Bejaia.

At the intersection of security and AI, my work focuses on intrusion detection systems, machine learning for cybersecurity, and post-quantum cryptography. I build things that matter — from marketplace security layers to drone network anomaly detection.

À l'intersection de la sécurité et de l'IA, mes travaux portent sur les systèmes de détection d'intrusion, le machine learning en cybersécurité et la cryptographie post-quantique. I'm also the co-founder of SOUK-DZ, a secure second-hand marketplace startup for Algeria, built with Next.js, Java Spring Boot, and an AI-powered security layer.

Je suis également co-fondatrice de SOUK-DZ, une startup algérienne de marketplace de seconde main sécurisée, construite avec Next.js, Java Spring Boot et une couche de sécurité basée sur l'IA.
